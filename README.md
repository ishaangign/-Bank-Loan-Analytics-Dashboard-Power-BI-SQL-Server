This project presents a comprehensive Power BI dashboard built using a Microsoft SQL Server database containing bank loan data.

📌 Project Overview

An end-to-end Business Intelligence solution designed to monitor loan performance, risk indicators, and borrower behavior.

📊 Report Pages

🔹 Summary

Key KPIs (Applications, Funded Amount, Received Amount)

Good vs Bad Loan Metrics

Interest Rate & DTI Analysis

MTD / YTD Performance

🔹 Overview

Loan distribution by category, grade, and purpose

Regional trends and time-based analysis

Interactive filtering and segmentation

🔹 Details

Transaction-level loan records

Borrower information

Drill-through analysis capabilities

🧠 Technical Implementation

SQL Server as primary data source

Data Modeling with relationships

Star schema design

DAX measures and calculated columns

Time Intelligence functions

Interactive report design

🛠️ Tools & Technologies

Power BI • Microsoft SQL Server • DAX • Data Modeling
